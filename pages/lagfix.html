> How to optimize FTB for better performance!

  • In the FTB Launcher, select "Options" from the top menu buttons. Give FTB more RAM. Don't go nuts here unless you know your computer has huge amounts of RAM. Setting this to 2GB should be ample.

  • Launch minecraft, then hit ESC to exit to the pause menu.
  • Select Video Settings and make the following changes:
    1. Set Graphics to "FAST"
    2. Turn Smooth Lighting OFF
    3. Set Particles to "Minimal"
    4. Set Render Distance to "Normal"
    5. Set Performance to "Max FPS"
    6. Turn Advanced OpenGL ON
    7. Set Use VSync to OFF (IMPORTANT)

These changes should help a LOT. If you are still having trouble, please let me know!
